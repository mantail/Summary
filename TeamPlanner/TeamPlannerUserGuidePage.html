User Guide


NOTE: this project is under development. The forms in both these extensions currently do nothing

Opening the program

1.Make sure you have an internet connection
2.Visit: https://team-b-great-meeting-scheduler.firebaseapp.com/

Logging in as a new user

1.Make sure you have a Google account, with an associated gmail address. If you do not have one, create one at https://accounts.google.com/SignUp?hl=en
2.Enter your gmail address and password into their respective text inputs, then click the ‘New User’ button. Your information will be appended to the database and you will be redirected to the Main Page.
⚠️.If an error message appears, ensure your information is correct, that you are connected to the internet, and that there are enough characters in your password. If all of these have been corrected and the error persists, the cause is likely an error on the backend.

Logging in as an existing user

1.Make sure you have both a Google account with an associated gmail address, and that it has been previously registered as a New User.
2.Enter your gmail address and password into their respective text inputs, then click the ‘Login’ button. Your information will be verified and you will be redirected to the Main Page.
If an error message appears, ensure your information is correct, and that you have previously registered as a New User using said information. If all of these have been corrected and the error persists, the cause is likely an error on the backend.

Logging out

1.Make sure you are logged in, and on any page except the Login Page.
2.Click the ‘Log Out’ button. You will be returned to the Login Page.

Creating a group

1.Make sure you are logged in and on the Main Page.
2.Click the ‘Create Group’ button. You will be redirected to the Group Creation Page.
3.In the labelled Inputs, type in the Meeting Name, Start Date, End Date, Meeting Length, and the Number of Attendees.
NOTE: The date format is MM/DD/YYYY.?
NOTE: The date system uses midnight as time boundaries. Eg, starting at 11/21/9999 and ending at 11/21/9999 selects the full day of 11/21 as the meeting window, 11/21/9999-11/25/9999 selects the entirety of 11/21, 11/22, 11/23, 11/24, and 11/25 as the meeting window.
NOTE: If you don’t know Number of Attendees in advance, input -1 to create a group of indeterminate size. A group of indeterminate size has no way to automatically calculate meeting times, so the user MUST use the method described in ‘Selecting a time without all responses’ to get final results.
4.Once your desired information is input, click the ‘Create Group’ button. The Group Key will be displayed in the bottom right corner.

Joining a group

1.Make sure you are logged in, on the Main Page, and have a valid Group Key.
2.Click on the ‘Join Group’ button. You will be redirected to the Group Joining Page.
3.Enter your Group Key and click the ‘Submit Key’ button. You will be redirected to the Main Page, where the group associated with the Group Key will be added to your Group List.
⚠️.If an error message appears or no new groups appear in the Group List, ensure your Group Key is correct. If this has been corrected and the error persists, the cause is likely an error on the backend.

Choosing a group

1.Make sure you are logged in, on the Main Page, and that you have already joined/created a group on this account.
2.From the Group List, labelled ‘Existing Groups’, select the desired group by clicking it. You will be redirected to the Choice Page.

Adding a timeslot or timeslots

Selecting a meeting time

1.Make sure you are logged in, are on the Choice Page for the desired group, and have received timeslot responses from all members.
2.From the best possible time(s) displayed, labelled ‘Meeting Times’, select a time by clicking it.
3.The event will be added to a Group Google Calendar, and an email notification will be sent to each member of the group with the event’s information.

Selecting a meeting time without all response

1.Make sure you are logged in and are on the Choice Page for the desired group.
2.To calculate meeting times without responses from all users, click the ‘Calculate Now’ button. The Choice Page will refresh, now showing the best possible time(s) in ‘Meeting Times’.
3.From the best possible time(s) displayed, labelled ‘Meeting Times’, select a time by clicking it.
4.The event will be added to a Group Google Calendar, and an email notification will be sent to each member of the group with the event’s information.
⚠️.THIS WILL BE THE BEST TIME(S) FOR ONLY THOSE WHO HAVE SUBMITTED RESPONSES. TO CALCULATE THE BEST TIME(S) FOR ALL MEMBERS, WAIT FOR ALL MEMBERS TO SUBMIT RESPONSES.
