Please create a group by inputing the start and end date of the window the meeting can be held at, the length of the meeting, and the number of people to attend, excluding youself. This page will return a key to you, copy this key and send it to all members you wish to partake in the meeting. If you do not know the number of attendees, input -1 to indicate this. All other fields must be filled out to the best of your knowledge.
